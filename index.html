    History Is Derived from the Land

  • set of 4 pieces
  • Wood stain and charcoal on paper
  • 2019

The full name of this piece is in four parts. The names given in parentheses are the shorthand names for each piece.

( Terrane | Export | Consumption | Culture )

If History is an account of power through time...

    ( Terrane )

  • 17 x 26 in.
  • detail below

...And if power through time is determined

by who has what resources and when...

    ( Export )

  • 17 x 26 in.

...And if resources are derived from rocks in the earth

and from the forests that grow from those rocks

and from the animals that grow from those forests...

    ( Consumption )

  • 17 x 26 in.
  • detail below

...History is an account of humans' exploitation of nature

for power and for the comfort of the powerful.

    ( Culture )

  • 17 x 26 in.
